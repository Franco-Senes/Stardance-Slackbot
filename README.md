# My first slack bot 

## Features
* /dsb-ping - check bot latency
* /dsb-help - list all of the available commands
* /dsb-catfact - get a random cat fact
* /dsb-joke - get a random joke

## How to set it up
* Make sure to have nodejs and install the necessary packages with `npm i @slack/bolt axios dotenv`
* Create a `.env` file in the main folder and put your tokens like this:
```env
SLACK_BOT_TOKEN=xoxb-your-bot-token
SLACK_APP_TOKEN=xoxe-xapp-your-app-token
```
* Also make sure you turned on socket mode and added the commands there too

## How to run
If you want to run it locally just use node index.js or whatever you named the file if it says it works
then you are ready to use it.
